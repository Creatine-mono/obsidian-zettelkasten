- **2.SSL/TLS**
    - SSL/TLS란?
        - SSL(Secure Sockets Layer)은 암호화 통신을 가능하게 하는 *보안 프로토콜*
        - 요즘은 **TLS**(Transport Layer Security)가 표준이며, SSL은 옛날 버전
    - HTTPS (Hypertext Transfer Protocol Secure)
        - HTTP +TLS
        - 웹사이트와 주고받는 데이터를 암호화
        - 평문 HTTP 대신 **HTTPS를 쓰면 TLS를 통해 암호화**
