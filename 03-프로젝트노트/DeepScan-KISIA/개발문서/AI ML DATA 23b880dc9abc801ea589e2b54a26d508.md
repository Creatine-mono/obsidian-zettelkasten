## 구현 필요 기능

- (필수) 변조 데이터 식별 기능 (개발 완료)
    - OIㅇ - 하나2three → 010 -123
    - 데이터 : 이름, 주민등록번호,전화번호
- (확장) 시계열 데이터 분석 기능
    - 내부 위협자 A는 1일차에 010, 2일차에 234, 3일차이 567 을 전송하였다
- (확장) 분할 데이터 검토
    - 내부 위협자 A는 010, B는 234, C는 567을 전송하였다
