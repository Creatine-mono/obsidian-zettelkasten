**1-4 코드짜기**
    
    
    <?php
    if ($_SERVER["REQUEST_METHOD"] == "POST") {
        $id = $_POST['id'] ?? '';
    
        $password = $_POST['password'] ?? '';
    
        if ($id === 'admin' && $password === 'admin1234') {
            echo "<p>로그인 성공</p>";
        } else {
            echo "<p>아이디 또는 비밀번호가 잘못되었습니다.</p>";
        }
    }
    ?>

아이디는 admin 비번은 admin1234로 설정하고 

맞게 적었으면 로그인성공 틀리면 아이디 또는 비밀번호가 잘못되었습니다.이런식으로 나오게 설정
